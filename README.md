Village est un projet développé en PHP utilisant la programmation orientée objet.
Il met en place différentes entités (héros, monstres, objets, combats, bâtiments) afin de structurer un système de jeu ou de simulation et de pratiquer les bases de la POO.

Objectifs

Appliquer les principes de la programmation orientée objet en PHP

Structurer un projet de manière modulaire

Comprendre les relations entre classes

Manipuler des entités complexes (héros, monstres, objets, combats)

Structure du projet

Building/ : gestion des bâtiments

Combat/ : logique de combat

Games/ : gestion des parties ou scénarios

Hero/ : classes liées aux héros

Items/ : gestion des objets

Monster/ : classes liées aux monstres

test/ : fichiers de test

index.php : point d’entrée du projet

Installation

Cloner le dépôt :

git clone https://github.com/tahirylebg/codePOO.git

Placer le projet dans un serveur local (XAMPP, WAMP, MAMP, etc.)

Lancer le projet :

php index.php

ou ouvrir index.php dans un navigateur
